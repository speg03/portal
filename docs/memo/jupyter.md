# Jupyter

## Magic Commands

refs: [Built-in magic commands - IPython](https://ipython.readthedocs.io/en/stable/interactive/magics.html)

* `%matplotlib inline`: matplotlibのグラフを出力セルに表示
* `%time`: 同じ行の式を評価したときにかかった時間を表示
* `%%time`: 同じセルの式を評価したときにかかった時間を表示
